pycharm远程linux开发和调试代码

Permalink: 2019-08-08 10:20:00+02:00 by 韩敏 in Linux tags: Linux

1.1 配置远程linux主机信息

参考博客：https://www.cnblogs.com/lei0213/p/7898301.html

1) 选择Tools--Deployment--Configuration

2) 这里选择SFTP就可以

3) 这里添加SFTP主机信息然后可以天机测试连接,输入用户名和密码

4) Local path配置为你windows本地的代码路径就可以,Deployment path on server 这个配置为你linux上代码的路径 　　　　　　

5) 这个时候通过Tools--Deployment--Browse Remote Host就可以看到你远程主机的信息

1.2 配置远程linux的python

1) 使用pycharm打开本地项目，找到项目环境路径　　

2）配置SSH Credentials (ssh凭证)

3）将python环境指定未刚刚配置的远程环境
