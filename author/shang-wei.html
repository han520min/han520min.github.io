Permalink: 2019-08-08 18:44:00+02:00 by 商伟 in my-shang tags: shang

支付宝支付

  • 安装模块
  pip install Crypto 
  pip install pycryptodome
  pip install python-alipay-sdk

  • 下载RSA工具

  • 将生成的公钥上传到支付宝沙箱的RSA2(SHA256)密钥(推荐）处

  • 设置公钥后会生成一个支付宝公钥，将其复制
